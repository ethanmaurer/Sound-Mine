## Installation Instructions

### Windows

1. Go to the [latest release](https://github.com/ethanmaurer/Sound-Mine/releases/latest)
2. Download the `.exe` file (e.g., `SoundMine Setup X.X.X.exe`)
3. Run the installer
4. Follow the installation wizard
5. Launch SoundMine from your Start menu or desktop shortcut

### macOS

1. Go to the [latest release](https://github.com/ethanmaurer/Sound-Mine/releases/latest)
2. Download the appropriate `.zip` file:
   - **Intel Macs**: Download `SoundMine-X.X.X-mac.zip`
   - **Apple Silicon (M1/M2/M3)**: Download `SoundMine-X.X.X-arm64-mac.zip`
3. Extract the archive
4. Move `SoundMine.app` to your Applications folder
5. Launch SoundMine from your Applications folder

**Note for macOS:** If you see a security warning when first opening the app:

- Right-click the app and select "Open"
- Click "Open" in the security dialog
- This is normal for apps not signed with an Apple Developer certificate

## System Requirements

- **Windows**: Windows 10 or later
- **macOS**: macOS 10.15 (Catalina) or later
- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 100MB free space

## Supported Audio Formats

SoundMine supports the following audio formats:

- MP3
- WAV
- OGG
- M4A
- AAC
- FLAC
- WMA

## Getting Started

1. Launch SoundMine
2. Click "Add Button" to add your first sound
3. Select an audio file from your computer
4. Click the button to play the sound
5. Use the "More options" menu (⋮) to access additional features

For more detailed instructions, open the app and click **More options → Help**.
