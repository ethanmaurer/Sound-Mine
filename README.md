# SoundMine

A simple, customizable desktop soundboard application for organizing and playing audio files. Perfect for theater productions, live events, or any situation where you need quick access to sound effects and audio clips.

## What is SoundMine?

SoundMine lets you:

- Create custom soundboards with multiple buttons
- Organize sounds into different scenes/tabs
- Assign keyboard hotkeys for quick playback
- Customize button colors, sizes, and labels
- Save and load your soundboard configurations

## Features

- **Multiple Scenes**: Organize sounds into different tabs for different productions or events
- **Keyboard Shortcuts**: Assign hotkeys to any button for instant playback
- **Customizable Layout**: Configure 1-10 rows of buttons with customizable sizes and colors
- **Drag & Drop**: Easily reorganize buttons by dragging them between rows
- **Volume Control**: Adjust volume for each button individually
- **Theater Mode**: Hide editing controls for a clean performance interface
- **Auto-Save**: Your configuration is automatically saved

## Supported Audio Formats

MP3, WAV, OGG, M4A, AAC, FLAC, WMA
